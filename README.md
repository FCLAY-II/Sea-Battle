# Морской бой
## О проекте:
Веб-приложение реализовано на WebSocket, что позволяет пользователям играть в многопользовательском режиме. Был реализован MVP, в котором пользователь может зарегистрироваться/авторизоваться с использованием JSON Web Token, создать игру, посмотреть статистику своих игр(победы, поражение, общее количество игр) и начать сам процесс игры. Были реализованы алгоритмы правильной расстановки кораблей и алгоритмы самой игры.
